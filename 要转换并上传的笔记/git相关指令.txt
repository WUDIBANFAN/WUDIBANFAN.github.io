操作步骤
	1. 先确认：你是想把当前目录关联到远程 Git 仓库，还是直接克隆整个仓库？
方案 A：全新克隆仓库（推荐）
退出当前目录，执行克隆命令：

cd ~/wangce_pytest
git clone 你的仓库地址
克隆完成后，进入仓库目录，再执行切换分支命令：

cd 仓库名
git fetch origin
git checkout tianxu_medium_smoke
方案 B：如果已经有代码，初始化并关联远程

# 1. 初始化git仓库git init
# 2. 关联远程仓库git remote add origin 你的仓库地址
# 3. 拉取远端所有分支git fetch origin
# 4. 切换到目标分支git checkout -b tianxu_medium_smoke origin/tianxu_medium_smoke
情况1：已经成功切换到分支

git pull
这条命令会自动拉取当前分支（tianxu_medium_smoke）远端最新代码并合并。
情况2：刚创建本地分支，还没有跟踪远程分支
先绑定远程追踪，再拉取代码：

git branch --set-upstream-to=origin/tianxu_medium_smoke tianxu_medium_smoke
git pull
情况3：干净拉取远端完整代码（覆盖本地改动，谨慎使用）

git fetch origin
git reset --hard origin/tianxu_medium_smoke
最简完整流程（一次性执行）

git fetch origin
git checkout tianxu_medium_smoke
git pull
